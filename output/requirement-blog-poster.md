# 요구사항 정의서

## 기본 정보
- 플러그인명: blog-poster
- 목적: 기술/IT 블로그 글을 주제 리서치부터 최종 Word 문서 출력까지 자동화하는 블로그 작성 에이전트
- 대상 도메인: 콘텐츠 제작 (기술/IT)
- 대상 사용자: 개인 기술 블로거
- 프로젝트 디렉토리: 현재 선택된 폴더 (blog-poster)

## 핵심기능
- 주제 리서치: 주제 키워드 기반 웹 검색, 기술 문서 조사, 트렌드 분석, 참고 자료 수집
- 글 작성/편집: 리서치 결과 기반 블로그 본문 작성, 구조화, 톤/스타일 조정, Word(.docx) 문서 출력
- SEO 최적화: 키워드 분석, 메타 태그 생성, 제목/소제목 최적화, SEO 점수 평가
- 이미지 생성: AI 모델(Gemini) 기반 블로그 삽화/썸네일 이미지 자동 생성

## 사용자 플로우
- Step 1. 주제 입력: 사용자가 블로그 주제(키워드) 입력
- Step 2. 리서치: 주제 관련 자료 조사 및 분석
- Step 3. 초안 작성: 리서치 결과 기반 블로그 초안 작성
- Step 4. SEO 최적화: 초안에 SEO 요소 적용 및 최적화
- Step 5. 이미지 생성: 본문 맥락에 맞는 삽화/썸네일 생성
- Step 6. 최종 문서 출력: 글 + 이미지 + SEO 정보를 Word(.docx) 문서로 출력

## 에이전트 구성 힌트
- researcher (MEDIUM): 웹 검색, 기술 문서 조사, 트렌드 분석, 참고 자료 수집
- writer (MEDIUM): 블로그 본문 작성, 구조화, 편집, Word 문서 생성
- seo-optimizer (MEDIUM): 키워드 분석, 메타 태그 생성, 제목 최적화, SEO 점수 평가
- image-creator (MEDIUM): AI 모델 기반 블로그 삽화/썸네일 이미지 생성

## 참고 공유 자원
- 참고 도구: context7 (기술 문서 검색), generate_image (AI 이미지 생성)
- 참고 샘플: abra (플러그인 구조 참고)

## 추가 요구사항
- 글 언어: 한국어
- 글 톤/스타일: 친근하고 캐주얼한 문체 (독자에게 친근하게 다가가는 스타일)
- 글 분량: 짧은 글 (1,000~2,000자)
- 출력 형식: Word(.docx) 문서
